# 📚 BookLink

A modern full-stack bookmark management application that allows users to save, organize, and share links with public and private visibility.

---

## 🚀 Features

- 🔐 User authentication (Signup / Login)
- 📌 Create, update, delete bookmarks
- 🌐 Public and private bookmarks
- 👤 Unique user handles (@username)
- 📄 Public profile page for shared bookmarks
- 🔒 Secure user data with proper access control
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS (optional)

### Backend
- Node.js
- Express.js
- TypeScript

### Database / Auth
- Supabase (PostgreSQL + Auth)
